# グループホーム入居・見学のご相談AI
# Group Home Consultation AI

---

## 概要 / Overview

**日本語**
福祉業界向けのAI入居相談システムです。グループホームへの入居・見学希望者に対して、AIがヒアリングから要約・通知まで一連の相談業務をサポートします。

**English**
An AI-powered consultation system for the welfare/care industry. This system supports the entire consultation workflow — from AI-driven interviews to summaries and notifications — for individuals seeking to move into or visit group homes.

---

## 使用技術 / Tech Stack

| 技術 / Technology | 用途 / Purpose |
|---|---|
| Python | バックエンド / Backend |
| Streamlit | Webアプリフレームワーク / Web App Framework |
| OpenAI API | AI会話・要約生成 / AI Conversation & Summarization |
| Google Sheets | データ保存・管理 / Data Storage & Management |
| LINE Messaging API | スタッフへの通知 / Staff Notifications |

---

## 機能一覧 / Features

**日本語**
- AIヒアリング：入居希望者の状況・ニーズをAIが対話形式でヒアリング
- AI要約：ヒアリング内容をAIが自動で要約・整理
- Google Sheets保存：相談内容をGoogleスプレッドシートに自動保存
- LINE通知：担当スタッフにLINEで相談内容をリアルタイム通知

**English**
- AI Interviewing: AI conducts structured interviews to gather information about the prospective resident's needs and situation
- AI Summarization: Automatically summarizes and organizes the interview content using AI
- Google Sheets Integration: Automatically saves consultation data to Google Sheets
- LINE Notifications: Sends real-time notifications to staff via LINE Messaging API

## 開発背景 / Background

**日本語**
福祉業界での8年間の現場経験をもとに開発しました。入居相談業務における情報収集・記録・連絡の煩雑さに課題を感じ、AIを活用した業務DX（デジタルトランスフォーメーション）を目的として本システムを構築しました。現場のリアルな業務フローを反映した設計が特徴です。

**English**
This system was developed based on 8 years of hands-on experience in the welfare industry. Recognizing the complexity of information gathering, documentation, and communication in the consultation process, this project was built with the goal of digitally transforming (DX) group home consultation workflows. The design reflects real-world operational processes from the field.
